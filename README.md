Навчальний Android-додаток для створення списку покупок, розроблений на Kotlin з використанням Jetpack Compose та Room Database.

Реалізований функціонал:
-Додавання нових елементів у список
-Відображення списку покупок
-Позначення елементів як куплених (Checkbox)
-Збереження даних у локальній базі даних (Room)
-Відображення списку після перезапуску додатку

Додаткові покращення:
-Реалізовано можливість видалення елементів зі списку

Труднощі:
-Налаштування роботи з Room Database
-Розуміння роботи Jetpack Compose
